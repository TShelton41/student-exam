# React Native + Expo Router — Exam Project

This exam project requires you to build a small React Native application using **Expo** and **Expo Router**. Your app must use a tab layout and include a separate detail screen inside the `app` directory. Follow all requirements carefully for full credit.

---

## Instructions

1. **Clone the repo**

   ```bash
   git clone https://github.com/TShelton41/student-exam.git
   cd student-exam
   ```
